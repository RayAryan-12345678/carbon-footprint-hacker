How the ML pipeline works

📥 Dataset (5,000 records)

Synthetic but realistic records generated using real IPCC emission factors. Each row has 12 input features and a ground-truth CO₂ value with 5% real-world noise.

🧠 Naive Bayes Classifier

GaussianNB classifies lifestyle profiles into Low / Moderate / High tiers. Used as one component of the final ensemble.

📈 Gradient Boosting Regressor

200-tree GradientBoostingRegressor predicts exact monthly CO₂. Achieves R²=0.933 and MAE=18.75 kg on held-out test data.

🔀 Ensemble Classifier

Final tier = 35% × NaiveBayes + 65% × GradientBoostingClassifier probability. 83.1% accuracy on test set.

Core calculation formula
# Carbon Footprint Hacker — ground truth formula total_CO2 = transport_co2 + electricity_co2 + heating_co2 + food_co2 + waste_co2 + shopping_co2 # Components: transport_co2 = km x emission_factor[mode] + (flights/12) x 255 electricity_co2 = kWh x grid_factor x 0.233 heating_co2 = heating_base x grid_factor food_co2 = diet_base x food_source x food_waste x 30 waste_co2 = kg_waste x 1.2 x recycling_factor shopping_co2 = monthly_spend x 0.43 # Carbon score (0-100, higher = greener): score = max(0, 100 - (total / 833) x 50)
